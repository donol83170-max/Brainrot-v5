# Core Loop — Brainrot Wheel Royale

> Statut : BROUILLON

## La Boucle Principale
1.  **L’Attente/Action** : Le joueur s'approche de la "Roue de Départ".
2.  **Le Spin** : Coût 0 (ou petit montant de Coins). Animation de rotation fluide.
3.  **Le Drop** : Un objet Brainrot apparaît avec un effet visuel lié à sa rareté.
4.  **La Décision** :
    *   **Garder** : L'objet va dans l'inventaire (Collection).
    *   **Vendre** : L'objet est converti instantanément en **Coins**.
    *   **Échanger** : L'objet est gardé pour être proposé à un autre joueur.

---

## Système de Rareté (Roue n°1)

| Rareté | Probabilité | Couleur | Exemples de "Brainrots" (Libres de droits/Memes) |
| :--- | :--- | :--- | :--- |
| **Normal** | 60% | Gris / Blanc | "Bruh" Sound, Noob Face, Basic Pizza |
| **Rare** | 20% | Bleu | Mewing Emoji, Blue Tie, Sigma Smile |
| **Mythique** | 10% | Violet | Gigachad Jawline, Pizza Tower Pose |
| **Légendaire** | 8% | Orange/Or | Skibidi Head, Golden Sigma |
| **Ultra** | 2% | Arc-en-ciel / Néon | Galaxy Sigma, Diamond Skibidi, Ultra Mewing |

---

## Mécanique de Spin
*   **Cooldown** : Par défaut 5 secondes entre chaque spin gratuit.
*   **Luck Boosts** : Possibilité d'utiliser des potions (obtenues via vente ou codes) pour augmenter les chances de Mythique+.
*   **Multi-Spin** : Débloquable plus tard (x3, x10) pour accélérer le grind.

## Collection & Badges
*   Chaque nouvel objet découvert remplit l'**Album Brainrot**.
*   Compléter une rareté entière (ex: tous les Normaux) donne un bonus permanent de +5% de vitesse de spin.
