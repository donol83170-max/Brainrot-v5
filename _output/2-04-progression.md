## Missions Quotidiennes

Reset chaque jour à minuit. 3 missions disponibles par joueur.

### Types de missions

| Difficulté | Exemple de mission | XP |
|-----------|-------------------|-----|
| **Facile** | Joue 2 matchs / Lance la balle 10 fois | +50 XP |
| **Moyenne** | Fais 3 kills / Gagne 1 match | +75 XP |
| **Difficile** | Libère-toi 2 fois depuis la prison / Attrape 3 balles | +100 XP |
| **Bonus** | Complète les 3 missions du jour | +150 XP |

**Max XP quotidien via missions : +375 XP** (sans compter les matchs)

### Pool de missions (rotation aléatoire)

| Mission | Difficulté |
|---------|-----------|
| Joue X matchs | Facile |
| Lance la balle X fois | Facile |
| Participe à un match avec des amis | Facile |
| Fais X kills | Moyenne |
| Gagne X matchs | Moyenne |
| Attrape X balles | Moyenne |
| Libère-toi X fois depuis la prison | Difficile |
| Sois MVP d'un match | Difficile |
| Gagne un match sans être éliminé | Difficile |
| Joue les 3 modes différents | Difficile |

## Courbe de progression

```
Vitesse de progression
  ▲
  │███████
  │███████████
  │████████████████
  │████████████████████████
  │████████████████████████████████
  │██████████████████████████████████████████
  └────────────────────────────────────────────> Niveau
   1   5   10   20   30   50          100

Rapide au début → progressivement plus long → endgame pour les dédiés
```

- **Niveaux 1-10** (~2-4h de jeu) : Progression rapide, récompenses fréquentes, le joueur découvre les cosmétiques
- **Niveaux 10-50** (~15-30h) : Rythme régulier, identité visuelle du joueur se forge, missions quotidiennes cruciales
- **Niveaux 50-100** (~50h+) : Pour les joueurs dédiés, cosmétiques exclusifs, statut "Légende" en jeu

## Prestige (Post-MVP)

Non implémenté au MVP. À prévoir pour la Version Complète :
- Condition : Atteindre le niveau 100
- On garde : tous les cosmétiques débloqués
- On repart : niveau 1, avec une étoile de prestige visible sur le profil
- Bonus : multiplicateur XP x1.1 par prestige, titre exclusif
